Ollama 커스텀 모델을 Mac에서 실행하는 방법
============================================

이 문서는 Windows에서 생성한 GGUF 모델 파일을 Mac으로 옮겨, Ollama에서 로컬로 실행하는 방법을 안내합니다.

---

### 1단계 (Windows PC): GGUF 파일 외장하드로 복사

먼저, Windows PC에서 생성한 GGUF 모델 파일을 외장하드 등 이동식 저장 장치에 복사합니다.

- 복사할 파일 경로: `models\llama31-8b-merged\gguf-model-f16.gguf`

---

### 2단계 (Mac): Ollama 설치 및 파일 준비

이제 Mac으로 이동하여 다음을 진행합니다.

1.  **Ollama 설치:**
    Mac에 Ollama가 설치되어 있지 않다면, 공식 홈페이지(ollama.com)에 방문하여 Mac용 버전을 다운로드하고 설치합니다.
    - 링크: https://ollama.com

2.  **파일 준비:**
    외장하드에 있던 `gguf-model-f16.gguf` 파일을 Mac의 찾기 쉬운 폴더(예: `Downloads` 폴더)로 복사합니다.

---

### 3단계 (Mac): Ollama에 커스텀 모델 등록

Mac에서 터미널(Terminal) 앱을 열고 아래 절차를 따릅니다.

1.  **Modelfile 생성:**
    GGUF 파일을 복사해 둔 폴더(예: `~/Downloads`)로 이동합니다. 그리고 해당 폴더에 `Modelfile`이라는 이름의 새 파일을 만들고, 아래 내용을 정확히 한 줄만 입력합니다.

    ```
    FROM ./gguf-model-f16.gguf
    ```
    (참고: `./`는 현재 폴더를 의미합니다. GGUF 파일과 Modelfile이 같은 폴더에 있어야 합니다.)

2.  **Ollama 모델 생성:**
    터미널에서 GGUF 파일과 `Modelfile`이 있는 폴더인지 확인한 후, 아래 명령어를 실행합니다. 'robot-agent'라는 이름으로 Ollama에 모델이 등록됩니다.

    ```bash
    ollama create robot-agent -f Modelfile
    ```
    (생성 과정에서 GGUF 파일을 Ollama의 저장소로 복사하므로 약간의 시간이 소요될 수 있습니다.)

---

### 4단계 (Mac): 모델 실행

모든 등록 과정이 끝나면, 이제 Mac 터미널에서 언제든지 아래 명령어로 파인튜닝된 나만의 모델을 실행할 수 있습니다.

```bash
ollama run robot-agent
```

이제 `>>>` 프롬프트가 나타나면 모델과 직접 대화를 시작할 수 있습니다.
